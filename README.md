# Miller Custom Works

First-draft static site for Miller Custom Works.

This repo reuses the old `MillerContracting` GitHub repository and replaces the old placeholder with a new proof-first contractor site draft.

## Current draft

- Homepage: `index.html`
- Projects page: `projects/index.html`
- First project detail page: `projects/custom-wood-counters/index.html`
- Design rationale: `docs/design-direction.md`

## Local preview

```bash
python3 -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173/
```

## Validation

```bash
python3 scripts/check_site.py
```
